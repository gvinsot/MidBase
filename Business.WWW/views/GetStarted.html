Welcome to SilverScript

SilverScript is a javascript Framework built to bring WPF/Silverlight binding syntaxes to javascript/typescript world.


Discover the easiest way to develop a website extensively using MVVM development pattern.
SilverScript is designed to reduce significantly the need of creating code by connecting
your interface direclty to datasources and bind your UI elements to your datasoucres.



XAML developers will find it easy to learn using syntax they well know.
Lets say you defined in your javascript code : node.attributes["data-context"]=myObject;
Then you can use the following on children nodes:
<Button data-binding="{Binding Path=name,Destination=Content}"> </Button>
Here the result will simply be the name of the object in data-context to be set as content of the button.


We also added many other possibilities like retrieving the data-context directly from the server :
<div data-binding="{Binding Path=name,Destination=Content}" data-context="/viewmodels/mycontroller"> </div>


The power of SilverScript comes from the fact that you can mix the approaches :
<div data-context="/viewmodels/mycontroller">
<button data-binding="{Binding Path=name,Destination=Content}" data-context="{DataBinding Path=subObject}"></button>
</div>
